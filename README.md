# Aquaponics-monitoring-system-with-esp32-Rust
ESP32 aquaponics monitor built in Rust. Tracks pH, ammonia, nitrite, nitrate, dissolved oxygen, water temp, water level, flow rate, air temp, and humidity. Monitors pump, heater, and flood/drain cycle state. Alerts on pH swings, ammonia spikes, temp extremes, low water, and pump failure.
